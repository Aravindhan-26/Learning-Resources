# Markdown Tutorials
### Text Decorations
- _italic_ - surround words with an underscore ( _ ). For example, \_this\_
- **Bold** - surround words with two asterisks( ** ).  \*\*Bold\*\*
- Try this **_Bold and Italics_** (Place asterisks outside )

### Headers
- # Header one - \#
- ## Header two - \#\#
- ### Header three - \#\#\#
Upto header 6
